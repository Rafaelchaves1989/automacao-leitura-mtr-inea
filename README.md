# Automação de Extração de Dados - Portal INEA (RPA & Web Scraping)

Este projeto foi desenvolvido em Python para otimizar o tempo operacional na auditoria de Manifestos de Transporte de Resíduos (MTR) do portal do INEA (Instituto Estadual do Ambiente).

## 🚀 Funcionalidades
* **Automação de Login e Navegação:** Acessa o portal de forma segura.
* **Download em Lote:** Identifica e baixa os arquivos PDF dos MTRs automaticamente.
* **Data Extraction (ETL):** Realiza a leitura dos arquivos PDF estruturados e extrai campos específicos de interesse.
* **Consolidação:** Exporta todos os dados minerados de forma limpa para uma planilha Excel.

## 🛠️ Tecnologias Utilizadas
* **Python** (Linguagem principal)
* **Selenium / Requests** (Para navegação e download)
* **PyPDF2 / PDFPlumber** (Para leitura e extração de texto dos PDFs)
* **Pandas / OpenPyXL** (Para manipulação de dados e geração do Excel)

## 📈 Impacto no Negócio
Eliminação de tarefas 100% manuais e repetitivas de download e digitação, reduzindo o tempo de auditoria do setor e mitigando erros humanos na transferência de dados.# automacao-leitura-mtr-inea
